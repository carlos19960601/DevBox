**JSON 转 Schema**

将 JSON 数据转换为 JSON Schema，用于数据验证和文档化。

* 实时转换
* 暂停转换模式
* Schema 验证支持
* 💎 专业版功能

![](./screen/json-to-schema.png)

**QuickType 代码生成**

从 JSON 数据生成多种编程语言的代码类型。

* 支持多种语言（JavaScript、TypeScript、Python、Go 等）
* 类型定义生成
* API 模型创建
* 💎 专业版功能

![](./screen/quicktype.png)

**Unix 时间戳转换**

在 Unix 时间戳和人类可读日期之间进行转换。

* 支持秒和毫秒
* 多时区支持
* ISO 8601 格式
* ✅ 免费功能

![](./screen/unix-timestamp.png)

**颜色转换器**

在不同颜色格式之间转换（HEX、RGB、HSL、CMYK）。

* 实时颜色预览
* 颜色选择器集成
* 透明度通道支持
* 💎 专业版功能

![](./screen/color-convertor.png)

**Cron 表达式解析器**

解析 Cron 表达式并可视化构建计划任务。

* 可视化 Cron 构建器
* 预设计划模板
* 下次运行时间预览
* 表达式验证
* 💎 专业版功能

![](./screen/cron.png)

**文本对比**

比较和可视化两个文本之间的差异。

* 并排对比视图
* 多种粒度模式（字符/单词/行）
* 💎 专业版功能

![](./screen/text-diff.png)
